# Silicon City Pet Hospital Website

A professional, responsive website for Silicon City Pet Hospital built with HTML, CSS, and JavaScript. The site features appointment booking, service information, and comprehensive pet healthcare information.

## Features

✨ **Responsive Design** - Fully responsive on all devices (mobile, tablet, desktop)
🏥 **Professional Layout** - Modern design inspired by leading pet hospital websites
📱 **Mobile Menu** - Hamburger navigation for mobile devices
📅 **Appointment Booking** - Easy-to-use appointment booking form
🔧 **Services Showcase** - Detailed service descriptions and benefits
👥 **Team Information** - Meet the veterinary team
💬 **Testimonials** - Real customer feedback
📞 **Contact Information** - Multiple ways to reach us
🌙 **Modern Styling** - Professional color scheme (blue & green)

## How to Use

1. **Opening the Website:**
   - Open `index.html` in your web browser
   - Or use a local server (Python: `python -m http.server`, Node.js: `http-server`)

2. **Navigation:**
   - Click on menu items to navigate between pages
   - Use the appointment booking button for quick access
   - Mobile users can use the hamburger menu

3. **Booking an Appointment:**
   - Fill out the appointment form with your details
   - Select your preferred date and time
   - Choose the service type
   - Submit the form

4. **Contact Information:**
   - Use the contact form on the contact page
   - Call the emergency line for urgent matters
   - Visit the hospital at the provided address

## Customization

### Update Hospital Information
- **Phone Numbers:** Replace `(080) XXXX-XXXX` throughout the site
- **Email:** Update `info@siliconpethospital.com`
- **Address:** Modify the Bangalore address
- **Social Media:** Update social media links in footer

### Styling
- Colors can be modified in `css/style.css` root variables
- Font changes can be made in the body CSS rule
- Spacing and sizing can be adjusted as needed

### Adding Images
- Place images in `assets/images/` directory
- Update image paths in HTML files
- Recommended formats: JPG, PNG, WebP
